# Amazon One-Page Clone (HTML & CSS)

A static one-page Amazon-inspired webpage created using HTML and CSS. This project focuses on front-end layout, visual design, and page structuring.

## Purpose
- Practice building a web page layout inspired by a real-world website
- Learn section structuring and styling techniques
- Strengthen HTML & CSS skills

## Features
- Header and navigation bar
- Product sections
- Footer design
- Clean and readable design

## Folder Structure
- `index.html`          → Main web page
- `style.css`           → CSS styling
- `screenshots/`        → Preview images of the page

## Technologies Used
- HTML
- CSS
